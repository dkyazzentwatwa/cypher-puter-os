# Cardputer Tarot

## Overview

Cardputer Tarot is an offline tarot app with a full 78-card symbolic deck, spreads, history, journaling, study mode, favorites, clarifiers, and SD-backed pull records.

## What It Does

- Draws single-card and multi-card spreads from a full tarot deck.
- Supports upright/reversed pulls with duplicate prevention inside a spread.
- Stores previous pulls, notes, favorites, settings, and study history on SD.
- Provides study mode, clarifier cards, card replacement, and filtered history.
- Uses generated/code-drawn symbolism rather than bundled copyrighted artwork.

## Controls

- `;`, `,`, `W`, `K`: move up. `.`, `/`, `S`, `J`: move down.
- `A`/`H`: previous card/page. `D`/`L`: next card/page.
- `Enter`, `Space`, or BtnA: select.
- `Backspace`, `Delete`, `Tab`, backtick, or `Q`: back.
- `R`: retry/favorite/toggle depending on screen. `T`: add or edit reflection note. `M`: pull actions. `X`: guarded delete.

## SD And Runtime Files

- SD card stores pull history, settings, journal notes, favorites, and study data.
- Cypher OS app binary: `/cypher-puter/apps/cardputer-tarot.bin`.

## Return To Cypher OS

Back/backtick from the main menu returns to Cypher OS when launched from the SD catalog build.
