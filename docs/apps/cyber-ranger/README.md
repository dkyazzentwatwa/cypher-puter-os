## Overview

Cyber Ranger mixes park patrol fiction with fake anomaly boards and menu-driven deck tools. You gather biological and fictional signal evidence to resolve cases.

## Goal Of A Run

Resolve park cases by combining field evidence with fictional signal-chain evidence. Keep stamina, battery, heat, clues, funds, and trust under control.

## Screen Guide

### STATION

STATION plans active cases across Pine, Lake, Ridge, Bog, Lodge, Road, Cave, and Meadow leads.

### PARK

PARK patrols field zones and finds biological evidence.

### SIGNAL

SIGNAL inspects fictional anomalies such as weather echo, tag glitch, sensor ghost, and bio trace.

### DECK

DECK uses fake tools: scan, map, tag, decode, log, shield, savebat, and compass. These are menu actions, not real device functions.

### GUIDE

GUIDE reveals cryptid states such as unknown, tracks, behavior, habitat, and protected finding.

### CASE

CASE resolves investigations that need both biological and fictional signal evidence.

### GEAR

GEAR upgrades scanner, mapper, decoder, tag reader, log tool, shield, saver, and compass.

## Core Loop

1. Choose a case at STATION.
2. Patrol PARK for biological clues.
3. Use SIGNAL and DECK for fictional anomaly evidence.
4. Read GUIDE entries.
5. Upgrade GEAR.
6. Resolve CASE when both evidence types are ready.

## Risk, Progress, And Endings

Battery and heat limit deck use. Stamina limits patrols. Trust affects better outcomes. Endings include park truth, science archive, town cover, corp leak, signal pact, and burned deck.

## Tips For First Run

- Do not run deck commands until battery is safe.
- Field evidence and signal evidence work together.
- Upgrade tools if repeated actions are too expensive.
- Heat recovery is worth it before case resolution.
- Trust helps the better endings.

## Offline Fictional Scope

All signal inspection, deck commands, guide states, and case systems are fictional offline table mechanics. Cyber Ranger does not use real sensors, radio, Wi-Fi, Bluetooth, GPS, network access, tracking, scanning, or device probing.
