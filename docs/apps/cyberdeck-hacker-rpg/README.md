## Overview

Cyberdeck RPG is a cyberpunk fantasy about taking fake jobs, running simulated node commands, upgrading tools, recovering fictional files, and managing heat.

## Goal Of A Run

Complete enough jobs and story progress before heat, trace, low battery, or deck damage ends the career. Strong runs upgrade tools before accepting the hottest jobs.

## Controls

## Screen Guide

### SAFEHOUSE

SAFEHOUSE lets you rest, read leads, patch up, and prepare.

### JOB BOARD

JOB BOARD lists fake contracts such as Glass Bank Sim, Transit Ghostbox, Clinic Mirror, Weather Feed, Servo Pawn, Underpass, Root Door, and Ledger Sim.

### NODE

NODE runs fictional commands such as scan, spoof, decrypt, inject, scrub, disconnect, trace map, and ghost fork. Each command changes table stats only.

### TOOLS

TOOLS upgrades scanner, decryptor, spoof chip, injector, trace map, blackbox, battery, and shield.

### FILES

FILES reviews ledger, clinic, weather, transit, ghost, civic, street, and crown story files.

### MARKET

MARKET buys cold CPU, battery stack, clean shell, spoof key, trace sink, and patch kit style offers.

### REP

REP handles faction leaning and ending pressure.

## Core Loop

1. Rest or read leads at SAFEHOUSE.
2. Pick a job.
3. Run NODE commands until success or disconnect.
4. Spend credits on tools and market upgrades.
5. Read files to push story flags.
6. Keep heat and trace manageable.

## Risk, Progress, And Endings

Trace and heat punish greedy play. Battery and deck condition limit how long you can stay in a node. Endings include clean root, rich burnout, street myth, ghost file, corp deal, and deck dead.

## Tips For First Run

- Disconnect before battery collapses.
- Upgrade battery or shield early.
- Do not inject every time; it is fast but risky.
- Read files to understand story pressure.
- Heat recovery is usually worth a turn.

## Offline Fictional Scope

All nodes, scans, spoofs, decrypts, injects, traces, files, and markets are compiled fictional table mechanics. Cyberdeck RPG does not hack, scan, connect to networks, access credentials, use Wi-Fi, use Bluetooth, use radio, open sockets, capture packets, or access the internet.
